Core Services API Documentation

Detailed technical specifications for the Core Services orchestration layer, including authentication patterns, rate limiting, and standard response schemas.

Bearer Authentication

All requests to the Core Services API must include a valid JSON Web Token (JWT) in the `Authorization` header. Tokens are generated via the `/auth/token` endpoint.

curl -X GET "https://api.alpha.com/v1/services" \
    -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
